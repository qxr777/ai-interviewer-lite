# AI 面试 Agent - 产品规格

## 核心功能

| 功能 | 要求 | 优先级 |
|------|------|--------|
| 简历提取 | 姓名、技能、经历、教育 | P0 |
| Schema 验证 | 输出符合预定义 Schema | P0 |
| 问题生成 | 基于简历技能生成 | P0 |
| 回答评估 | 分数 + 反馈，方差 < 10 | P0 |
| 多轮对话 | 3-5 轮 | P0 |
| 报告生成 | 总分 + 各轮分数 + 简历 | P0 |

## 质量指标

| 指标 | 阈值 |
|------|------|
| 简历置信度 | > 0.7 |
| Schema 验证 | 100% 通过 |
| 评分方差 | < 10 |
| 响应时间 | < 10 秒/轮 |
